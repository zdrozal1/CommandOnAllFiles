# Workspace Formatter

A VS Code extension to format all files in a workspace or specific folders.

## Features

- **Format All Workspace Files:** Formats every supported file in your current workspace.
- **Format All Files in Folder:** Right-click any folder in the Explorer and select "Format All Files in Folder" to format its contents.

## Commands

- `Workspace Formatter: Format All Workspace Files` (`workspace-formatter.formatAll`)
- `Workspace Formatter: Format All Files in Folder` (`workspace-formatter.formatFolder`)
